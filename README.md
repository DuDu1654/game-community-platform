# 🎮 游戏社区平台

一个为 2W+ 活跃 QQ 群用户打造的游戏社区平台，整合论坛、游戏数据查询、实时聊天等功能。

## ✨ 项目特色

- **用户基础**：已有 2W+ 活跃游戏玩家用户（来自 QQ 群）
- **技术栈先进**：采用现代化全栈技术栈
- **微服务架构**：支持高并发、可扩展部署
- **实时交互**：支持 WebSocket 实时聊天、游戏数据同步
- **边缘计算**：利用 K8s 实现边缘节点部署，降低延迟

## 🏗️ 技术架构

### 前端

- Vue 3 + TypeScript
- Vite + Pinia + Vue Router
- TailwindCSS + Element Plus
- WebSocket + WebRTC

### 后端

- Node.js + Express
- TypeScript
- PostgreSQL + Redis
- JWT 认证 + RBAC 权限控制

### 部署

- Docker 容器化
- Kubernetes 编排
- GitHub Actions CI/CD
- 边缘节点部署
- 监控告警（Prometheus + Grafana）

## 🚀 快速开始

### 环境要求

- Node.js 18+
- Docker 20+
- PostgreSQL 15+
